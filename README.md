# GISの実習用オープン教材の提供サイト(試験公開版)

## 本サイトについて
　本サイトは、科学研究費補助金　基盤研究(A) 「GISの標準コアカリキュラムと知識体系を踏まえた実習用オープン教材の開発」プロジェクトにおいて開発した、**「GISオープン教材(試験公開版)」** の提供を目的としたものです。
